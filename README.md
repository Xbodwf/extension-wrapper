# Extension Wrapper

Based on [Scratch Extension Template](https://github.com/FurryR/scratch-ext)


Add a CCW extension loader for Turbowarp or PenguinMod.

## Usage

* Download JS File
* Load extension in your Scratch project
* Load CCW Extensions then

## Development

```bash
git clone https://github.com/xbodwf/extension-wrapper.git
```

```bash
cd extension-wrapper
pnpm install 
# npm install
# yarn install
```

For production:
```bash
pnpm build
```
And files will be generated at the folder `dist`

For development:
```bash
pnpm start
```

Then load extensions from localhost url.
